# Manual-Digital-CT
Integrated SEIQR + Contact Tracing Dashboard using R Shiny
This interactive dashboard will allow users to input different variables to find disease spread, cost effectivness of various contact tracing methods, as well as historical demographic data for digital contact tracing methods.

Outputs include: I Curve, Area Under the Curve of I Curve, Peak Day of I curve People reached, Cost per Method, Cost breakdown by Component, Breakeven Point
